How to Add a Product
1. Go to "Add Product" from the sidebar.
2. Fill in product name, price, quantity, and select a category.
3. Upload an image if desired.
4. Click "Save" to add the product.
Managing Categories
1. Navigate to "Categories" in the admin panel.
2. Click "Add Category" to create a new one.
3. Use the edit or delete buttons for existing categories.
